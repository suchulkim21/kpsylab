# MNPS 분석 정확도 95% 달성 연구: 질의·응답만으로 높이는 방법

## 1. 목표와 제약

- **목표**: 분석 정확도를 **강제(인위적 상한/하한) 없이**, **질의·응답 결과만**으로 95%에 근접하게 만든다.
- **제약**: `Math.max(95, ...)` 같은 인위적 상한을 쓰지 않는다. 응답 품질(정직성, 일관성)이 좋을 때만 95%가 나오도록 한다.

---

## 2. 현재 정확도 공식 요약

| 요소 | 설명 | 범위 |
|------|------|------|
| **기본점** | 80 | 고정 |
| **검증 보너스** | v1~v4(정직·일관·역문·실제반영) 평균 → 0~20 | 0 ~ +20 |
| **일관성 패널티** | 4개 trait 점수 범위(spread) < 20이면 “평탄 응답”으로 간주 | 0 ~ -20 |
| **최종** | 80 + 보너스 - 패널티, 50~99로 클램프 | 50 ~ 99 |

- **95%가 나오는 조건**: 검증 보너스 +15 이상, 일관성 패널티 0  
  → 예: 검증 평균 75% 이상 + spread ≥ 20 이면 80+15=95.

즉, **“정직·일관하게 답하고, 4개 특성에 어느 정도 차이를 보이는 응답”**일 때 이미 95%가 나올 수 있는 구조다.  
연구 방향은 **이 조건을 “질의·응답만”으로 더 잘 충족시키고, 오탐(방어/조작 응답)은 그대로 낮게 유지**하는 것이다.

---

## 3. 질의·응답만으로 95%에 도달하는 방법

### 3.1 검증 문항 확대 (우선 권장)

**현재**: 검증 4문항(v1~v4) — 정직 1, 일관 1, 사회적 바람직성 역 1, 실제반영 1.

**문제**: 문항 수가 적어 검증 점수의 **측정 신뢰도**가 낮음. 우연히 한두 문항에서 낮게 답해도 전체 보너스가 크게 떨어질 수 있음.

**개선**:
- 검증 문항을 **6~8문항**으로 확대.
  - 정직/솔직성: 2문항 (예: “나는 항상 정직하게 답변했다”, “어려운 질문에도 솔직히 답했다”)
  - 일관성: 2문항 (예: “선택한 답변은 서로 일관된다”, “앞뒤가 맞게 답했다”)
  - 사회적 바람직성(역문): 2문항 (예: “좋게 보이려고 답을 고른 적이 있다”, “사회적으로 바람직한 답을 선택하려 했다”)
  - 실제 반영: 1~2문항 (예: “내 답변은 실제 생각을 반영한다”)
- **역문항**은 6−점수로 역코딩 후 평균에 포함.
- **효과**: 검증 점수의 안정성 증가 → 정직·일관한 응답자일수록 **검증 보너스가 안정적으로 +15 이상** → **질의·응답만으로 95% 도달 가능성 상승**. 방어적/조작적 응답자는 검증 점수가 낮아 그대로 95 미만 유지.

**구현 시**: `questions.ts`에 검증 문항 2~4개 추가, `scoreDarkNature`의 `validationScores`에서 새 ID(v5~v8 등)를 읽어 평균 계산에 포함.

---

### 3.2 검증 점수 높을 때 일관성 패널티 완화

**현재**: spread(4개 trait max−min) < 20이면 무조건 패널티.  
**한계**: 일부 응답자는 **진짜로** 4개 특성 점수가 비슷할 수 있음. 이때 검증 문항에서 “정직·일관하게 답했다”고 했는데도 spread만으로 패널티를 주면 95%가 잘 나오지 않음.

**개선**: “질의·응답만”의 정보를 더 쓰기 위해, **검증 점수가 높을 때는 spread 패널티를 줄인다**.

- 예시 1: `consistencyPenalty = rawPenalty * (1 - validationScore/100)`  
  - 검증 100% → 패널티 0, 검증 50% → 패널티 절반.
- 예시 2: `validationScore >= 70`이면 `consistencyPenalty`를 50% 감면(또는 0).
- **효과**: “정직·일관” 자기보고가 높은 사람은 spread가 다소 낮아도 패널티가 줄어 **질의·응답만으로 95%**가 나오기 쉬움. 검증 점수 낮은 사람은 기존처럼 패널티 유지 → 조작/방어 응답은 여전히 낮은 정확도.

**구현 시**: `darkNatureScoring.ts`의 `consistencyPenalty` 계산에서 `validationScore`를 사용하는 조건 추가.

---

### 3.3 Trait 내부 일관성(문항 간 일관성) 반영

**아이디어**: 같은 trait 내 4문항 점수가 **서로 비슷하면** (분산 낮음) “그 trait에 대해 일관되게 답한 것”으로 보고, **소량 보너스**를 주어 정확도를 올린다. 모두 “보통”만 고르면 분산은 낮지만, 그 경우는 이미 **spread(4개 trait 간)** 가 낮아 일관성 패널티로 걸러짐.

**방법**:
- trait별로 4문항 점수의 **표준편차(SD)** 또는 **역분산** 계산.
- SD가 “적당한 구간” (예: 0.3~1.2, 1~5 척도 기준)이면 “차별화되면서도 일관된 응답”으로 간주해 **+3~5점** 보너스.
- SD가 극단적으로 낮음(거의 같은 답) → 보너스 없음 또는 소량 패널티(기존 spread 패널티와 중복 최소화).
- **효과**: “문항만의 결과”(같은 trait 내 응답 패턴)로 정확도가 올라가며, **강제 상한 없이** 95%에 기여.

**구현 시**: `scoreDarkNature`에서 문항별 답변을 trait별로 묶어 SD 계산 후, `analysisAccuracy`에 소량 가산.

---

### 3.4 일관성 쌍 문항(동의어 문항) 도입

**아이디어**: 같은 구인을 **다른 표현**으로 묻는 문항 쌍을 2~4쌍 넣는다. 두 답의 차이가 작을수록 “일관된 응답”으로 보고 보너스, 차이가 크면 패널티.

- 예: n1 “나는 때때로 남보다 낫다고 생각한다” / n1b “나는 자신이 다른 사람들보다 우수한 편이라고 느낀다” (같은 trait, 비슷한 의미).
- **일관성 지표**: 쌍별 |답1−답2|의 평균. 평균이 임계값 이하면 +5, 크면 -5 등.
- **효과**: **질의·응답만**으로 “진짜 일관된 사람”을 구분해 정확도에 반영 → 95%는 정직+일관 응답자에게 더 자주 나옴.

**구현 시**: `questions.ts`에 쌍 문항 추가, `scoreDarkNature`에 쌍별 차이 기반 보너스/패널티 로직 추가.

---

### 3.5 검증 문항 가중 및 세분화

**현재**: v1~v4를 동일 가중 평균.  
**개선**: 역할별로 가중치를 둠.

- **정직/실제반영**(v1, v4): 가중 1.2 — “정직하게 답했다”, “실제 생각을 반영한다”가 정확도와 직결.
- **일관성**(v2): 가중 1.0.
- **사회적 바람직성 역문**(v3): 가중 1.2 — 역코딩 후, 이 점수가 낮을수록(바람직성 편향 적음) 보너스에 더 반영.

가중 평균 후 0~100으로 정규화해 `validationScore`로 사용.  
**효과**: “질의·응답만”의 정보를 더 잘 반영해, 정직·일관·바람직성 낮음이 분명한 응답자일수록 검증 보너스가 커져 95%에 기여.

---

## 4. 정리: 우선순위별 권장

| 순위 | 방법 | 난이도 | 효과 | 비고 |
|------|------|--------|------|------|
| 1 | 검증 문항 6~8문항으로 확대 | 중 | 높음 | 검증 점수 신뢰도↑ → 95% 자연 도달 |
| 2 | 검증 점수 높을 때 spread 패널티 완화 | 낮 | 중상 | 정직 보고자 중 spread 낮은 경우 95% 가능 |
| 3 | 검증 문항 가중(정직·역문 강조) | 낮 | 중 | 기존 4문항만으로도 정확도 공식 개선 |
| 4 | Trait 내부 일관성(SD) 보너스 | 중 | 중 | “문항만”으로 95% 기여 |
| 5 | 일관성 쌍 문항 2~4쌍 추가 | 높음 | 높음 | 문항 설계·채점 로직 확장 필요 |

---

## 5. 결론

- **강제로 95%를 올리지 않고**, “질의·응답만”으로 95%에 가깝게 만들려면:
  1. **검증 문항 확대**로 정직·일관·바람직성 측정을 안정화하고,
  2. **검증 점수가 높을 때** trait spread 패널티를 완화하며,
  3. (선택) **검증 가중**, **trait 내부 일관성**, **쌍 문항**을 단계적으로 도입하는 것이 좋다.

이렇게 하면 “정직하고 일관되게 답한 응답”에 대해서는 **자연스럽게 95% 근처**가 나오고, 방어/조작 응답은 기존처럼 낮은 정확도로 유지된다.
