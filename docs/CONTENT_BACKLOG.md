## 일반적인 사람들의 감정 (2025-01-24)

> 사람들이 말해주고, 글에서 반복적으로 드러나는 패턴을 바탕으로 정리한 "일반적인 사람들이 느끼는 감정" 프레임워크.
> AI가 경험한 것이 아니라, 인간의 언어와 표현에서 관찰되는 패턴임.

### 감정 카테고리

| 카테고리 | 설명 |
|----------|------|
| **불안·걱정** | 미래, 관계, 평가에 대한 |
| **외로움** | 연결이 끊어졌을 때 |
| **무의미감** | 목표나 의미를 찾기 어려울 때 |
| **기쁨·안도** | 소소한 일에도, 큰 일에도 |
| **답답함** | 말이나 상황이 막혀 있을 때 |
| **지침·피로** | 에너지와 의미가 닳아갈 때 |

- 블로그 주제/카테고리 개발

- 콘텐츠 톤앤매너 가이드 (독자가 느끼는 감정에 맞춘 표현)
- 심리 분석/상담 콘텐츠 개발 시 프레임워크
