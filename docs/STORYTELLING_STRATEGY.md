# KPSY LAB 스토리텔링 전략

리포트를 '개인의 성전(Scripture)'으로 만들어, 사용자가 서비스에 머물며 **경외감**을 느끼게 합니다.

---

## 1. 결과 → 여정(Journey)

**기존**: "당신은 A타입입니다"  
**변경**: "당신은 현재 A라는 상태를 통과해 B라는 가능성으로 나아가는 중입니다"

### 구현

- `journeySummary`: `adaptModule1`에서 타입별 NEXT_POSSIBILITY 매핑
  - A → "완성보다 충분함을 아는 여유"
  - B → "자기 확신 기반의 경계"
  - C → "감정 수용과 함께하는 실행력"
  - D → "의도적 멈춤과 깊은 성찰"

### 영웅의 여정

- **출발**: 검사 전 혼란
- **시련**: 마주한 취약점(그림자)
- **귀환**: 마스터 벡터 기반 새로운 삶의 지침

## 2. 데이터 블록 → 인생의 키워드

| 데이터 | 키워드 | 예시 문구 |
|--------|--------|----------|
| 높은 불안 | 섬세한 안테나 | "당신의 불안은 고통이 아니라, 주변의 미세한 변화를 감지하는 예민한 안테나입니다." |
| 재검사 변화 | 심리의 파동 | "지난달보다 높아진 외향성은 당신이 세상과 연결되기로 결심한 용기의 증거입니다." |
| 벡터 불일치 | 내면의 대화 | "두 결과의 차이는 당신 안의 '신중한 나'와 '도전하는 나'가 대화하고 있음을 뜻합니다." |

**구현**: `lib/services/narrativeMapping.ts`

## 3. The Sage Engine (서사 자동 생성)

**역할**: 수치를 개인화된 심리 서사로 변환

### 가이드라인

- **동반자 어조**: "우리는 당신의 내면에서 이런 움직임을 발견했습니다"
- **은유**: 뿌리 깊은 나무, 흐르는 강물, 단단한 원석
- **미래 가치**: G3 시대·지능 자산으로서의 강점
- **희망적 마무리**

**구현**: `lib/prompts/sageEngine.ts`

## 4. 나만의 심리 지도 (문장형 로그)

**예시**  
"2025년 12월, 당신은 불안을 마주했고(M1), 2026년 2월, 그 불안을 통제력으로 바꾸어(재검사), 지금은 통합된 마스터 벡터를 통해 단단한 자아의 중심을 세우고 있습니다."

**구현**: `lib/services/psychologicalMap.ts` → `generatePsychologicalMapLog()`
