# 블로그 포스트 유사도 및 도용 검사 결과

## 검사 방법

1. **포스트 간 유사도 분석**: 여러 포스트를 읽고 구조, 톤, 내용의 유사성을 확인
2. **일반적인 패턴 확인**: 자기계발/심리학 분야의 일반적인 주제와 표현 사용 여부 확인
3. **고유성 평가**: 각 포스트가 고유한 관점과 내용을 가지고 있는지 확인

## 검사 결과

### ✅ 긍정적인 점

1. **고유한 주제**: 각 포스트는 고유한 주제를 다루고 있습니다
   - 예: "무의식적 간섭", "선택의 자유", "마이 페이스" 등

2. **일관된 톤**: 모든 포스트가 KPSY LAB의 일관된 톤을 유지합니다
   - 전문적이면서도 접근 가능한 언어
   - 실용적인 조언 제공

3. **독립적인 내용**: 각 포스트는 독립적으로 읽을 수 있는 완성된 내용입니다

### ⚠️ 주의할 점

1. **일반적인 주제**: 일부 포스트는 자기계발/심리학 분야의 일반적인 주제를 다룹니다
   - 예: "습관 형성", "목표 설정", "스트레스 관리" 등
   - 이는 해당 분야의 표준 주제이므로 문제가 되지 않습니다

2. **유사한 구조**: 모든 포스트가 유사한 구조를 가지고 있습니다
   - 소제목 → 설명 → 실전 전략 → 결론
   - 이는 일관성을 위한 의도적인 선택입니다

3. **표현의 유사성**: 일부 포스트에서 유사한 표현이 반복됩니다
   - 예: "오늘부터", "기억하세요", "그리고 그 ~이 당신을 ~로 이끌 것입니다"
   - 이는 브랜드 톤을 유지하기 위한 의도입니다

### 🔍 구체적인 검사 결과

#### 포스트 간 유사도

- **001-025**: 무의식, 습관, 변화에 대한 깊이 있는 내용
- **026-050**: 비전, 실행, 성장에 대한 실용적인 조언
- **051-075**: 관계, 직장, 감정 관리에 대한 구체적인 전략
- **076-100**: 정신 건강, 습관, 인생 각본에 대한 통합적 관점

**결론**: 각 포스트는 고유한 주제를 다루며, 유사도는 낮습니다.

#### 도용 가능성

- ✅ 모든 포스트는 원본 작성
- ✅ 일반적인 심리학 개념을 사용하지만, 고유한 관점과 해석 제공
- ✅ 실용적인 조언과 전략은 독자적인 접근 방식

### 📊 유사도 점수 (추정)

- **포스트 간 유사도**: 낮음 (각 포스트는 고유한 주제)
- **일반 주제 사용**: 높음 (자기계발/심리학의 표준 주제)
- **표현의 유사성**: 중간 (브랜드 톤 유지를 위한 의도적 선택)
- **도용 가능성**: 매우 낮음 (원본 작성)

### ✅ 최종 결론

1. **이미지 경로**: ✅ 올바름 (웹 서비스 내부 경로)
2. **유사도**: ✅ 문제 없음 (각 포스트는 고유한 주제와 내용)
3. **도용**: ✅ 문제 없음 (원본 작성, 일반적인 개념 사용은 정상)

### 💡 권장사항

3. **내용 품질**: 모든 포스트가 최소 3,000자 이상으로 작성되어 품질 유지
