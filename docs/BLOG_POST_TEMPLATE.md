# 블로그 포스트 작성 템플릿

## 사용 방법

1. 관리자 대시보드 (`/admin/dashboard`) 접속
2. "새 포스트 작성" 버튼 클릭
3. 아래 형식으로 작성된 내용을 복사하여 붙여넣기

---

## 템플릿 형식

### 제목
```
[주제에 맞는 제목]
```

### 작성자
```
KPSY LAB
```

### 날짜
```
YYYY-MM-DD (오늘 날짜)
```

### 태그
```
태그1,태그2,태그3 (2-5개, 쉼표로 구분)
```

### 이미지 경로
```
/images/blog/topic_XXX.jpg (XXX는 001~120 사이 숫자)
```

### 내용 (HTML 형식)
```html
<h3>섹션 제목 1</h3>
<p>본문 내용을 여기에 작성합니다. 최소 3,000자 이상으로 상세하게 작성해야 합니다.</p>

<h3>섹션 제목 2</h3>
<p>추가 내용을 작성합니다.</p>
<ul>
  <li>리스트 항목 1</li>
  <li>리스트 항목 2</li>
  <li>리스트 항목 3</li>
</ul>

<h3>섹션 제목 3</h3>
<p>더 많은 내용을 작성합니다. <strong>중요한 부분은 강조</strong>할 수 있습니다.</p>
<p>여러 문단으로 나누어 작성하면 가독성이 좋아집니다.</p>
```

---

## HTML 작성 규칙

### 사용 가능한 태그
- `<h3>` - 섹션 제목 (h1, h2는 사용 금지)
- `<p>` - 문단
- `<ul>`, `<ol>`, `<li>` - 리스트
- `<strong>` - 강조
- `<em>` - 기울임
- `<blockquote>` - 인용
- `<br>` - 줄바꿈

### 사용 금지 태그
- `<h1>`, `<h2>` - 사용하지 마세요
- `<div>`, `<span>` - 가능하면 피하세요
- `<script>`, `<style>` - 보안상 사용 금지

## 예시: 완성된 포스트

### 제목
```
왜 우리는 같은 실수를 반복하는가: 무의식적 간섭의 이해
```

### 작성자
```
KPSY LAB
```

### 날짜
```
2025-01-15
```

### 태그
```
무의식,습관,변화,심리학
```

### 이미지 경로
```
/images/blog/topic_042.jpg
```

### 내용
```html
<h3>반복되는 실수의 심리학</h3>
<p>우리는 종종 같은 실수를 반복합니다. 다이어트를 시작했다가 며칠 만에 포기하고, 운동 계획을 세웠다가 한 주도 지나지 않아 중단합니다. 이런 패턴이 반복되는 이유는 무엇일까요?</p>

<h3>무의식이 만드는 자동 반응</h3>
<p>심리학자들은 우리의 행동 중 상당 부분이 무의식에 의해 결정된다고 말합니다. 특히 익숙한 상황에서는 뇌가 자동으로 기존 패턴을 재생합니다. 이는 에너지를 절약하기 위한 뇌의 효율적인 메커니즘입니다.</p>

<p>하지만 문제는 이 자동 반응이 때로는 우리에게 해로운 결과를 가져온다는 점입니다. 예를 들어, 스트레스를 받으면 자동으로 과식을 하거나, 불안할 때 손톱을 물어뜯는 등의 습관이 형성됩니다.</p>

<h3>변화를 방해하는 심리적 장벽</h3>
<p>변화를 시도할 때 우리는 여러 심리적 장벽에 부딪힙니다:</p>
<ul>
  <li><strong>저항</strong>: 익숙한 것이 편안함을 주기 때문에 변화를 거부합니다.</li>
  <li><strong>두려움</strong>: 새로운 행동의 결과를 예측할 수 없어 불안해합니다.</li>
  <li><strong>자기 효능감 부족</strong>: "나는 변할 수 없다"는 믿음이 변화를 막습니다.</li>
</ul>

<h3>무의식 패턴을 인식하는 방법</h3>
<p>무의식적 패턴을 바꾸기 위해서는 먼저 그것을 인식해야 합니다. 다음 질문들을 스스로에게 던져보세요:</p>
<ul>
  <li>언제 이 행동을 반복하게 되나요?</li>
  <li>어떤 감정이나 상황이 이 패턴을 유발하나요?</li>
  <li>이 행동의 즉각적인 보상은 무엇인가요?</li>
  <li>장기적으로 어떤 결과를 가져오나요?</li>
</ul>

<h3>변화를 위한 실전 전략</h3>
<p>무의식적 패턴을 바꾸는 것은 쉽지 않습니다. 하지만 불가능하지는 않습니다. 다음 전략들을 시도해보세요:</p>

<p><strong>1. 환경 재설계</strong>: 나쁜 습관을 유발하는 환경을 바꿉니다. 예를 들어, 다이어트를 하려면 냉장고에 건강한 음식만 넣어두세요.</p>

<p><strong>2. 작은 변화부터</strong>: 큰 변화보다는 작은 변화가 지속 가능합니다. 하루에 10분씩 운동하는 것부터 시작하세요.</p>

<p><strong>3. 습관 스택킹</strong>: 기존 습관에 새로운 습관을 연결합니다. 커피를 마실 때마다 물 한 잔을 마시는 것처럼요.</p>

<h3>결론: 인식에서 변화로</h3>
<p>같은 실수를 반복하는 것은 우리의 의지력 부족이 아닙니다. 무의식이 만든 자동 반응 패턴 때문입니다. 이 패턴을 인식하고, 환경을 바꾸고, 작은 변화부터 시작한다면 우리는 분명히 변화할 수 있습니다.</p>

<p>변화는 하루아침에 일어나지 않습니다. 하지만 매일 작은 선택을 통해 우리는 새로운 패턴을 만들어갈 수 있습니다. 오늘부터 자신의 무의식적 패턴을 관찰해보세요. 그것이 변화의 첫 걸음입니다.</p>
```
