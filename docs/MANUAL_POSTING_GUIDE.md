# 수동 블로그 포스트 작성 가이드

## 📝 작성 프로세스

### 1단계: 주제 선택
- Phase 1 주제 목록 (`docs/topics_phase1.txt`)에서 오늘 작성할 주제 선택
- 또는 새로운 주제 제안

### 2단계: AI에게 요청
```
"오늘의 블로그 포스트 주제: [주제명]"
또는
"[주제명]에 대한 블로그 포스트 작성해줘"
```

### 3단계: 작성된 내용 확인
- AI가 제공한 HTML 형식의 포스트 내용 확인
- 필요시 수정 요청

### 4단계: 관리자 대시보드에 입력
1. `/admin/dashboard` 접속
2. "새 포스트 작성" 버튼 클릭
3. AI가 제공한 내용을 각 필드에 입력:
   - 제목
   - 내용 (HTML)
   - 작성자: KPSY LAB
   - 날짜: 오늘 날짜 (YYYY-MM-DD)
   - 태그: 쉼표로 구분 (2-5개)
   - 이미지 경로: `/images/blog/topic_XXX.jpg` (001~120)
4. 저장 클릭

---

## 📋 작성 요청 예시

### 예시 1: 간단한 요청
```
오늘의 주제: "왜 우리는 같은 실수를 반복하는가: 무의식적 간섭의 이해"
```

### 예시 2: 상세한 요청
```
주제: "왜 우리는 같은 실수를 반복하는가: 무의식적 간섭의 이해"
태그: 무의식,습관,변화
이미지: topic_042.jpg
```

### 예시 3: 여러 개 요청
```
오늘 3개 포스트 작성해줘:
1. "왜 우리는 같은 실수를 반복하는가: 무의식적 간섭의 이해"
2. "당신의 선택은 정말 당신의 것인가? 무의식적 충동 분석"
3. "'나'를 안다는 것의 착각: 메타인지의 중요성"
```

## ✅ 작성 완료 체크리스트

- [ ] 제목이 명확하고 매력적인가?
- [ ] 내용이 최소 3,000자(약 1,000단어) 이상인가?
- [ ] HTML 형식이 올바른가? (h3, p, ul, li 등)
- [ ] h1, h2 태그가 없는가?
- [ ] 태그가 2-5개인가?
- [ ] 이미지 경로가 올바른 형식인가?
- [ ] 날짜가 올바른가?
