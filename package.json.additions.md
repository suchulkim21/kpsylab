# 추가 필요한 패키지

다음 패키지들을 설치하면 추가 기능이 활성화됩니다:

## 필수 (E2E 테스트)
```bash
npm install --save-dev axe-playwright
```

## 선택적 (Sentry)
```bash
npm install --save @sentry/nextjs
```

## 선택적 (Web Vitals)
```bash
npm install --save web-vitals
```

## 설치 명령어 (모두)
```bash
npm install --save-dev axe-playwright
npm install --save @sentry/nextjs web-vitals
```
